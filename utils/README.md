# General Tools & Utilities (`utils/`)

This directory houses general-purpose scripts, automated workflows, and tools developed to support my development, simulation, and repository management processes.

## Key Utilities
* **`workspace.sh`**: A self-contained bash script used to securely compress, encrypt, and bundle project workspaces into portable, standalone self-extracting scripts.
* **Automation Helper Scripts:** Short scripts (Bash, Python, or Makefiles) designed to clean environment directories, parse log files, or automate recurring tasks.
